# INTRODUCTION
It is just easy to integrate Kafka with SpringBoot. Just add the dependency as following:
```xml
<dependency>
            <groupId>org.springframework.kafka</groupId>
            <artifactId>spring-kafka</artifactId>
        </dependency>
```

1. using kafkaTemplate to act as a message producer
2. using @KafkaListener to act as a message consumer
3. you can specify groupId in application.properties like `spring.kafka.consumer.group-id=ei`

The most important, you should use `spring.kafka.bootstrap-servers=localhost:9094` in `application.properteis` 
to specify the ip and port about kafka.
